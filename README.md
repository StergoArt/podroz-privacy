# Polityka Prywatności — Podróż Dookoła Zdrowia

To repozytorium hostuje **politykę prywatności** aplikacji *Podróż Dookoła Zdrowia* — edukacyjnej gry mobilnej dla dzieci.

## Hostowana wersja (publiczny URL)

**https://stergoart.github.io/podroz-privacy/**

Ten URL należy podać w panelach App Store Connect (Apple) oraz Google Play Console.

## TL;DR

- Aplikacja jest **offline-first** — nie zbiera żadnych danych osobowych
- Wszystkie informacje (postęp w grze, imię, statystyki) są zapisywane **wyłącznie lokalnie** w pamięci urządzenia
- **0 trackerów / 0 analytics SDK / 0 in-app purchases**
- Zgodność: COPPA (USA), GDPR-K (UE)

Pełna treść: [`index.md`](./index.md) lub https://stergoart.github.io/podroz-privacy/
